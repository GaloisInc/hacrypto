PAEQ

Jump to: navigation, search

PAEQ [peik] (Parallelizable Authenticated Encryption based on Quadrupled AES) is an authenticated encryption scheme with the following features: 

- Variable key, nonce, tag length;
- Simple permutation-based design;
- Security level up to 160 bits, equal to the key size;
- Parallelism;
- Online processing;
- Optional protection to nonce-misuse;
- Quick tag update.

Full version of the PAEQ paper, published at ISC 2014.
